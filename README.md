# 🚀 TrendSage AI – Social Media Trend Analyzer

TrendSage AI is an intelligent analytics chatbot that tracks, clusters, and summarizes emerging trends across social media platforms like Twitter, Reddit, and YouTube. It helps businesses and analysts identify viral topics, monitor brand sentiment, and extract actionable insights in real time.

---

## 🎯 Objective

To automatically detect trending discussions, analyze sentiment, and generate business-ready insights using AI-powered clustering and summarization.

---

## 💡 Key Features

- 🔍 **Trend Detection** – Identifies viral and emerging topics in real time  
- 🧠 **Semantic Clustering** – Groups similar discussions using SentenceTransformers  
- 😊 **Sentiment Analysis** – Tracks positive, negative, and neutral sentiment  
- 📊 **Engagement Insights** – Measures topic popularity using likes, shares, and comments  
- 🤖 **AI Summarization** – Generates concise summaries of top discussions  
- 💬 **Chatbot Interface** – Interactive querying and drill-down analysis  
- 📈 **Dashboard Ready Output** – Structured JSON for visualization tools  

---

## 🛠 Tech Stack

- Python  
- SentenceTransformers (for embeddings)  
- LLM (for summarization & insights)  
- Pandas / NumPy  
- Streamlit (optional dashboard)  
- APIs: Twitter, Reddit, YouTube  

---

## ⚙️ How It Works

1. Collect daily social media posts via APIs  
2. Clean and preprocess the data  
3. Generate embeddings for each post  
4. Cluster similar content into topics  
5. Analyze sentiment distribution  
6. Detect trending patterns  
7. Summarize insights using LLM  
8. Output results in chatbot + dashboard format
